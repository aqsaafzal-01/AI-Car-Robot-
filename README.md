## 📌 Overview

**BOB (formerly SARAS)** is a DIY AI-powered robotics platform designed to demonstrate how intelligence can be embedded into physical machines using embedded systems, sensors, and real-time decision-making.

It bridges the gap between **software intelligence and real-world robotics** through a modular, expandable architecture.

---

## 🚀 Key Features

- 🚗 Autonomous navigation system  
- 👁️ Real-time computer vision (Pi Camera)  
- 🎙️ Voice-based interaction (input/output)  
- 🧠 Intelligent decision-making logic  
- 📡 Multi-sensor obstacle detection  
- 🔋 Dual power architecture (logic + motors)  
- 🧩 Modular 3-layer robotic design  
- 🔧 Fully expandable for AI/ML upgrades  

---

## 🧠 System Architecture

### 🔹 Perception Layer
- Pi Camera → Visual input  
- Ultrasonic Sensors → Distance detection  
- Microphone → Voice input  

### 🔹 Processing Layer
- Raspberry Pi 4 Model B → Central brain  
- AI decision-making engine  
- GPIO hardware control system  

### 🔹 Action Layer
- DC Motors → Movement  
- L298N Motor Driver → Motor control  
- Speaker → Audio output  
- Display → User interface

## 🛠️ Assembly Steps

1. Build chassis frame  
2. Install motors and wheels  
3. Mount motor driver  
4. Connect motor wiring  
5. Install power system  
6. Mount Raspberry Pi  
7. Connect GPIO wiring  
8. Add ultrasonic sensors  
9. Install camera module  
10. Attach display, mic, speaker  
11. Power system setup  
12. Run full system tests  

---

## 🧪 Working Principle

BOB works in a continuous intelligent loop:

> **Sense → Process → Decide → Act → Respond**

1. Sensors collect real-world data  
2. Raspberry Pi processes inputs  
3. AI logic determines actions  
4. Motors and modules execute response  
5. System adapts in real time
